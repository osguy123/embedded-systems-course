## Course Source:
https://www.youtube.com/playlist?list=PLPW8O6W-1chwyTzI3BHwBLbGQoPFxPAPM

- C programming for microcontrollers
- Hardware interfacing

- JTAG/SWD Debugging
- Datasheet Navigation
- Interrupts
- Real-Time Operating Systems
- Object-Oriented Programming
- State Machines
